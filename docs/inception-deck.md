# 需求背景

由台湾光谱资讯公司出品的，类大富翁类型，三国题材游戏，本系列只到富甲天下 5。

因为光谱不做富甲天下 6，所以自制一个。

# 目标对象

本游戏以富甲天下系列粉丝（简称情怀党）、喜欢轻松 Q 版休闲游戏的玩家为目标对象。

力求做成简明、欢快、不怎么烧脑的休闲三国战旗游戏。
